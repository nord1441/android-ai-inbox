# AI Inbox プライバシーポリシー

最終更新日: 2026-05-09

本プライバシーポリシーは、Android アプリ「**AI Inbox**」（パッケージ名: `uk.nordtek.aiinbox`、以下「本アプリ」）の取り扱う情報について説明します。

## 1. 結論を最初に

- 本アプリが取り込んだユーザーのデータ（共有されたテキスト、画像、OCR 結果、要約結果、タグ、人物・場所・URL・予定の抽出結果、検索クエリなど）は、**いかなる場面でも外部のサーバーに送信されません**。
- 処理はすべて端末内で完結します。クラウド推論は使いません。
- 外部通信はモデル / モジュールのダウンロードのみで、いずれもダウンロード方向です。アップロードは伴いません。
- クラッシュ解析・利用統計・広告 SDK は組み込んでいません。

---

## 2. 取得しないデータ

本アプリは以下を**取得しません**。

- 個人情報（氏名、メールアドレス、電話番号、住所など）
- 端末識別子（Android ID、広告 ID など）
- 位置情報
- 利用統計・クラッシュレポート・パフォーマンスメトリクス
- 連絡先・カレンダー・通話履歴・SMS

---

## 3. 端末内で扱うデータ

ユーザーが本アプリに取り込んだ以下の情報は、すべて**端末ローカルにのみ**保存されます。

- 他アプリから「共有」メニュー経由で送られたテキスト・画像
- 画像から端末内 OCR で抽出されたテキスト
- 端末内 LLM が生成した要約・タイトル・カテゴリ・タグ
- 抽出された人物名・場所・URL・予定情報
- 検索クエリ
- アプリ内設定値

これらはアプリのプライベート領域（`/data/data/uk.nordtek.aiinbox/`）に保存され、他アプリからアクセスできません。

### 暗号化

- データベース本体: **SQLCipher (AES-256)** で全文暗号化
- 添付画像ファイル: AndroidX Security の **EncryptedFile (AES-256-GCM)** で暗号化
- 暗号化鍵は Android Keystore で保護され、端末から取り出せません

---

## 4. 外部通信について（透明性のため）

本アプリの外部通信は以下の **ダウンロードのみ** であり、ユーザーが入れた情報のアップロードは一切行いません。

### 4.1 モデル重みのダウンロード

| 項目 | 内容 |
|---|---|
| 通信先 | Hugging Face（`huggingface.co`） |
| 内容 | Gemma 4 (E2B / E4B) の LiteRT-LM 形式モデル重み |
| サイズ | 数百 MB 〜数 GB |
| タイミング | ユーザーが設定画面でモデル DL を実行したとき |
| 方向 | ダウンロードのみ |

ダウンロード時に Hugging Face に送られる情報は、HTTP リクエストに付随する標準的なメタデータ（IP アドレス、User-Agent、ダウンロード対象のファイル名など）のみです。本アプリが取り込んだユーザーデータは送信されません。

### 4.2 ML Kit Text Recognition モジュールのダウンロード

| 項目 | 内容 |
|---|---|
| 通信先 | Google Play Services |
| 内容 | ML Kit Text Recognition の Latin / Japanese スクリプトモジュール |
| サイズ | 合わせて数〜十数 MB |
| タイミング | ユーザーが画像取り込みなどで OCR を初回利用したとき |
| 方向 | ダウンロードのみ |

これらのダウンロードはすべて HTTPS で行われ、ユーザーデータの送信は伴いません。

### 4.3 オフライン動作

上記のダウンロードが完了した以降は、**機内モードでも全機能（取り込み・OCR・要約・分類・検索・削除）が動作します**。インターネット接続なしで日常利用できる設計です。

---

## 5. 第三者への提供

本アプリは、ユーザーの情報を**いかなる第三者にも提供しません**。

- 広告 SDK は組み込んでいません
- Analytics SDK は組み込んでいません
- クラッシュレポート SDK（Firebase Crashlytics 等）は組み込んでいません

---

## 6. ユーザーによるデータの削除

### アプリ内からの個別削除

ホーム画面で各アイテムをスワイプ / 長押しから削除できます。削除は Undo 付きのソフト削除 → 確定後にファイル実体まで完全に消去されます。

### アプリのアンインストールによる全消去

本アプリをアンインストールすると、`/data/data/uk.nordtek.aiinbox/` 配下のすべてのデータ（暗号化 DB、添付画像、モデル重み、設定値、暗号化鍵）が Android OS によって削除されます。

### Markdown 同期フォルダ

ユーザーが設定で「ローカル/外部 Markdown 同期」フォルダを指定している場合、そのフォルダにエクスポートされた Markdown ファイルはアンインストール後もユーザー指定の場所に残ります。これらのファイルはユーザー自身がファイルマネージャ等で管理してください。

---

## 7. 権限について

本アプリが要求する Android 権限は以下です。

| 権限 | 用途 |
|---|---|
| `INTERNET` | モデル / OCR モジュールのダウンロードのみ。ユーザーデータの送信には使用しません |
| `POST_NOTIFICATIONS` | モデル DL 進捗・要約完了・予定検出の通知表示 |
| `FOREGROUND_SERVICE` / `FOREGROUND_SERVICE_DATA_SYNC` | LLM 推論中・モデル DL 中にプロセスが OS に kill されないようにするため |

過剰な権限は要求しません。位置情報、連絡先、カメラ、マイク、SMS などの権限は使いません。

---

## 8. 子供のプライバシー

本アプリは特定の年齢層を対象としていませんが、データの外部送信を一切行わないため、子供を含むあらゆるユーザーにとって安全に利用できる設計です。13 歳未満の子供から個人情報を意図的に収集することはありません。
